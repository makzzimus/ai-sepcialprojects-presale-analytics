# AI Presale Assistant для рекламных спецпроектов

## О инструменте 
Internal web-инструмент, предназначенный для поддержи Проджект и Креативных Менеджеров в сейлзхаузах и рекламных агенствах
Продукт анализирует клиентский бриф (в формате текста или документа), выявляет пробелы и риски, формирует структурированные вопросы для клиента, помогает менеджеру подготовиться к креативному шторму с более чётким пониманием контекста и возможных ограничений
